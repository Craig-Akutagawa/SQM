# 🛡️ 软件质量与管理 · 复习系统

> 📖 交互式复习网站，支持刷题、进度追踪和学习打卡

## ✨ 功能特性

- **📚 10讲结构化复习笔记** - 含知识点精讲与考点提炼
- **🧩 交互式选择题刷题** - 单选/多选，即时评分，解析详情
- **✅ 学习进度打卡** - Checklist 记录每个知识点掌握情况
- **📊 刷题进度面板** - 实时显示做题数量与正确率
- **💾 自动保存进度** - 基于 `localStorage`，刷新不丢失
- **🗂️ 选项卡分区** - 期末真题分「客观题」与「主观题」两栏

## 📁 项目结构

```
.
├── 复习资料/                  # 源文件
│   ├── 00_课程导读与大纲.md
│   ├── 01_第一讲_...md
│   ├── ...
│   ├── generate_static_pages.py  # 静态页面生成器
│   └── html_version/             # 自动生成（勿手动编辑）
├── 软件质量管理-0X-..._files/    # 课程配图
├── .github/workflows/deploy.yml  # 自动部署工作流
└── index.html                    # 入口重定向页面
```

## 🛠️ 本地开发

```bash
# 编辑 .md 文件后，重新生成 HTML
python 复习资料/generate_static_pages.py

# 直接在浏览器打开
复习资料/html_version/00_课程导读与大纲.html
```
